cordova的环境配置及简单使用

发表于 2020-03-24 | 分类于 Hybird App
字数统计：603 字 | 阅读时间 ≈ 2 分钟
一、安装cordova

cordova的安装需要node的环境，所以电脑需要安装node环境，然后执行全局安装命令。

npm install -g cordova
安装完毕，可以使用下面这个命令查看 cordova 的版本。
cordova -v
如果有显示版本号，说明已经全局安装成功了。

二、配置安卓SDK和java jdk环境
1、安卓SDK
  1. 下载 Android Studio

    地址：https://developer.android.google.cn/studio/

    打开 android studio 点击 file/settings/appearance&behavior/system settings/android SDK 在这里可以看到你的SDK地址，把它复制下来。
  2. 配置 SDK 环境
    打开我的电脑–属性–高级–环境变量

    新建系统变量 ANDROID_HOME，变量名：ANDROID_HOME ，变量值为：刚才复制的内容，也就是SDK的路径。

    在 Path 环境变量加追加：

    %ANDROID_HOME%\tools 和 %ANDROID_HOME%\platform-tools
  3. 打开命令行，输入 adb ，出现信息即表示环境配置成功。
2、java jdk

cordova好像最高只支持 java 8，安装更高的版本，会一直提示报错，无法找到环境。

java 8 的下载地址：https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html
根据自己的电脑系统版本选择对应的安装包下载。

配置环境：

  1. 打开我的电脑 – 属性 – 高级 – 环境变量
  2. 新建系统变量 JAVA_HOME，变量名： JAVA_HOME，变量值为：（刚刚安装时java jdk安装时的路径）例如：C:\Program Files\Java\jdk1.8.0_171
  3. 新建系统变量 CALSSPATH，变量名： CALSSPATH，变量值为：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
  4. 选择系统变量中的 Path 环境变量，在后面追加 %JAVA_HOME%\bin 和 %JAVA_HOME%\jre\bin
  5. 在cmd命令行输入 javac -version ，出现版本号，说明环境配置成功。
三、开始
1、初始化项目

创建项目的指令：
cordova create testdemo com.yourname.testdemo TestDemo
ps:创建一个 名字为testdemo的文件夹，项目名字为TestDemo,包名为com.yourname.testdemo的Cordova项目。

2、添加平台

添加安卓平台
cordova platform add android
添加ios平台
cordova platform add ios

3.打包

cordova build android

4、查看、删除平台

查看平台

cordova platform list
删除平台

cordova platform rm 平台名字

5、插件的操作指令

删除插件

cordova plugin rm 插件名字
添加插件

cordova plugin add 插件名字
