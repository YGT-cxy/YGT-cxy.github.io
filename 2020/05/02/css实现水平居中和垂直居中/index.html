css实现水平居中和垂直居中

发表于 2020-05-02 | 分类于 css
字数统计：1.3k 字 | 阅读时间 ≈ 6 分钟

水平居中

行内元素

利用 text-align: center; 设置父级元素的样式。

块级元素
固定宽度

  1. 利用 width 和 margin: 0 auto; 实现。

  2. 利用 position: absolute; 和 margin-left: -width / 2; 实现，给父元素添加 position: relative; 。

不固定宽度

  1. 利用 table 和 margin: 0 auto; 实现。

  2. 利用 display: inline-block; 和 text-align: center; 实现。

  3. 利用 position: absolute; 和 transform: translateX(-50%); 实现。

  4. 利用 flex 弹性布局的 justify-content: center; 实现。

垂直居中

行内元素
  1. 通过设置 line-height 实现垂直居中，仅限于文本类。
块级元素
固定高度

  1. 通过设置父级元素 绝对定位 ，子级元素 相对定位 ，top、right、bottom、left皆为0 和 margin: auto; 实现。
    ps:父级元素和子级元素需要设置固定的高度

  2. 通过 flex 布局和 margin: auto; 实现。
    ps: 父级和子级元素需要设置固定的高度

不固定高度

  1. 通过设置父级元素 position: relative;， 子级元素 position: absolute; top: 50%; transform: translateY(-50%); 实现。

  2. 通过设置父级元素 display: table;，子级元素 display: table-cell; vertical-align: middle;

  3. 通过 flex 布局的 align-items: center; 实现。

总结

  1. 实现水平居中的方式：

    • 行内元素通过利用 text-align: center; 设置父级元素的样式。
    • 块级元素固定宽度利用 width 和 margin: 0 auto; 实现。
    • 块级元素固定宽度利用 position: absolute; 和 margin-left: -width / 2; 实现，给父元素添加 position: relative; 。
    • 块级元素不固定宽度利用 table 和 margin: 0 auto; 实现。
    • 块级元素不固定宽度利用 display: inline-block; 和 text-align: center; 实现。
    • 块级元素不固定宽度利用 position: absolute; 和 transform: translateX(-50%); 实现。
    • 块级元素不固定宽度利用 flex 弹性布局的 justify-content: center; 实现。

  2. 实现垂直居中的方式：

    • 行内元素通过设置 line-height 行高等于高度，仅限于文本类。
    • 块级元素固定高度通过设置父级元素 绝对定位 ，子级元素 相对定位 ，top、right、bottom、left皆为0 和 margin: auto; 实现，父级和子级元素需要设置固定的高度。
    • 块级元素固定高度通过 flex 布局和 margin: auto; 实现，父级和子级元素需要设置固定的高度。
    • 块级元素不固定高度，通过设置父级元素 position: relative;， 子级元素 position: absolute; top: 50%; transform: translateY(-50%); 实现。
    • 块级元素不固定高度，通过设置父级元素 display: table;，子级元素 display: table-cell; vertical-align: middle;。
    • 块级元素不固定高度，通过设置父级元素flex 布局的 align-items: center; 实现。
