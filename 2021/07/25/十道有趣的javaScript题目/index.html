答案：b

解析：稀疏数组，[,,] 中间的元素为 empty ，这种我们就称为稀疏数组，我们也可以通过类似 new Array(2) 的方式创建稀疏数组。那为什么不是 4 而是 3 呢？这个跟 JavaScript 的尾后逗号 有关。MDN 中的解析如下：

尾后逗号 （有时叫做 “终止逗号”）在向 JavaScript 代码添加元素、参数、属性时十分有用。如果你想要添加新的属性，并且上一行已经使用了尾后逗号，你可以仅仅添加新的一行，而不需要修改上一行。这使得版本控制的代码比较（diff）更加清晰，代码编辑过程中遇到的麻烦更少。

换个角度思考：

在对象最后一个属性后面添加逗号，和不添加逗号的效果一致，其实数组也是一样，数组的实质也是对象，从这点来看，也就不难理解上面的答案了。
