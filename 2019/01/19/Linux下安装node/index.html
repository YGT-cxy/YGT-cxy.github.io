Linux安装node

wget下载node

wget https://nodejs.org/dist/v10.9.0/node-v10.9.0-linux-x64.tar.xz

解压压缩包

tar xf node-v10.9.0-linux-x64.tar.xz

进入解压目录

cd node-v10.9.0-linux-x64/

检查是否安装完毕

./bin/node -v – 需要进入到解压目录才能运行命令

建立node命令的软链接，方便调用命令

ln -s node的安装目录/bin/npm /usr/local/bin/ – 建立npm的软链接
ln -s node的安装目录/bin/node /usr/local/bin/ – 建立node的软链接

forever – 使node服务在后台运行

npm install -g forever

forever – 使node后台运行js文件，起到类似守护进程的功能

建立forever软链接，方便调用命令

ln -s node的安装目录/bin/forever /usr/local/bin/

使用forever启动项目

进入到项目的目录，启动项目：forever start 项目文件

停止forever的所有运行项目

forever stopall

查看forever运行的项目

forever list

查看forever的访问日志

forever logs
