hexo设置动态背景

发表于 2019-01-26 | 分类于 hexo
字数统计：386 字 | 阅读时间 ≈ 1 分钟

开启动态背景

修改_config.yml文件里的canvas_nest配置改为true

动态背景脚本加载路径 - CDN

canvas_nest和canvas_nest_nomobile配置上路径，也可以使用本地文件

动态背景脚本使用本地文件

  1. 进入next主题目录下的source/lib
  2. 获取背景脚本模块文件， 下载https://github.com/theme-next/theme-next-canvas-nest source/lib/canvas-nest并解压到source/lib
    (可以使用git命令：$ git clone https://github.com/theme-next/theme-next-canvas-nest source/lib/canvas-nest)
  3. 修改_config.yml配置里的配置项

自定义背景配置项

  • color：线条颜色, 默认: '0,0,255'；三个数字分别为(R,G,B)
  • opacity：线条透明度（0~1）, 默认: 0.5
  • zIndex：背景的z-index属性，css属性用于控制所在层的位置, 默认: -1
  • count：线条的总数量, 默认: 99
